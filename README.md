## 📝 Overview

Educore is a **modern educational landing page** built with **React** and **Tailwind CSS**.  
It is fully responsive and features a clean, interactive UI with smooth animations, perfect for showcasing courses, instructors, and FAQs.

---

## ✨ Features

- Fully **responsive design** for desktop, tablet, and mobile
- **Smooth scrolling navigation** to different sections
- **Accordion sections** for course modules and FAQs
- **Instructor profiles** with stats and brand logos
- **Pricing and enrollment card** with modern UI
- **Advanced contact form** with floating labels and icons
- Modern **glassmorphism** styling effects
- Fully deployable on **Vercel**

---

## 🛠️ Tech Stack

- **React.js** – Component-based UI library
- **Tailwind CSS** – Utility-first styling framework
- **Lucide Icons** – SVG icon library
- **Vercel** – Hosting and deployment
